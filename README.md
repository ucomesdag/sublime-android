# Sublime Android #

This is a pluging for Android development with [Sublime Text 3](http://www.sublimetext.com/3).

It automates basic tasks like:

* creating new projects
* refactor strings
* snippets
* build
* build on save
* create certificates
* install apk's
* uninstall apk's
* run on device
* run adb shell
* run logcat
* access sdk tools

It should work on windows/osx/linux.
